Structural Type System

you can use structures of things to define typing

                    
                        interface Point {
                            x: number;
                            y: number;
                        }
                        function logPoint(p: Point) {
                            console.log("${p.x}, ${p.y}");
                        }

                        const point = { x: 12, y: 26 };
                        logPoint(point); // OK
