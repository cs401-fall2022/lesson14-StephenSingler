Types by Inference

Typescript is able to determine typings of objects by inference:

                    
                        let helloworld="helloworld"//this is string

Here we see that the variable helloworld is inferred to be a string

Defining Types

One way to do so is to use an interface:

                    
                        interface user = {
                            name: string;
                            id: number;
                        }

Then you would declare an object of the interface

                    
                        const user: User = {
                            name: "joe"
                            id: 1;
                        }

Typescript will throw an error if done incorrectly.

Composing Types

Unions

You can make your own custom types with unions:

                    
                        type WindowStates = "open" | "closed" | "unlocked";
                        type PositiveOddNumbersUnderTen = 1 | 3 | 5 | 7 | 9;

Generics

Generics or void types are also a possibility in Typescript:

                    
                        type StringArray = Array<string>;
                        type ObjectWithNameArray = Array<{ name: string }>;

Structural Type System

You can use the structures of things to define typing. This is another method of inferrential type determination.

                    
                        interface Point {
                            x: number;
                            y: number;
                        }
                        function logPoint(p: Point) {
                            console.log("${p.x}, ${p.y}");
                        }

                        const point = { x: 12, y: 26 };
                        logPoint(point); // OK

Here we see the inferred relationship between the Point interface and the variable point.
